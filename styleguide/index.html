Introduction

Logo

What logo should I use?

There are two colour types, and a few file formats to choose from, when you need a logo:

For printing, use the CMYK version, and either an EPS or PDF format to ensure quality.

For viewing on a screen (Website, Email, Social Media post etc) use the RGB version. Where possible, use the SVG version to retain quality.

However, some applications and contexts don’t allow you to use a EPS, PDF or SVG file (such as Mailchimp), so in these cases a PNG will give you the best results. Never use a JPEG or BMP as the quality will be poorer.

Colour

The RTSL colour palette uses two main colours (RTSL Red and RTSL Blue), with tints of these used for backgrounds. RTSL Accent is used sparingly to highlight elements, while RTSL Soft Black is used for text

Colour values shown are CSS variables and CMYK, HSL and HEX Values.

Typography

RTSL uses the open-source font HK Grotesk:

Embedding HK Grotesk

As HK Grotesk is open-source, it's license allows us to embed the font in documents such as PDFs.

The brand font can be embedded within Word and Powerpoint documents if the TrueType version is used. Under Preferences > Output and Sharing > Save make sure 'Embed fonts in the file' is checked.

For use online, .woff versions are also available. Embed via the hk-grotesk.css stylesheet.

Basic typography rules

Heading 1: 3.875rem

Heading 2: 1.625rem

Heading 3 & 4: 1rem, Uppercase


Typography for RTSL starts with a base font size of 16px. The baseline grid/line height is set to 26px (16 × 1.6). Headings are sized similarly using a scale of 1.6. Type size is set to automatically enlarge as the viewport expands using this rule:

	    	
      	  html { font-size: calc(1rem + 0.15vw);}

For web use, the basic typography rules can be found in the typography.css file, which is separate from any style rules used just for this guide.
